Bootstrap is a versatile toolkit that includes HTML, CSS, and JavaScript
tools for creating and developing web pages and applications.

Bootstrap is simple to use and anyone with a basic understanding of HTML and CSS can get started. Features that adapt to phones, tablets, and desktops: Bootstrap's responsive CSS adapts to phones, tablets, and desktops. A mobile-first strategy: Mobile-first styles are built into the Bootstrap framework. Bootstrap 4 is compatible with all modern browsers, including Chrome, Firefox, Internet Explorer 10+, Edge, Safari, and Opera.

The .container class creates a fixed-width container that is responsive. The .container-fluid class creates a full-width container that spans the entire viewport width.

The default font size in Bootstrap 4 is 16px, with a line-height of 1.5. The default font family is "Helvetica Neue," which includes Helvetica, Arial, and other sans-serif fonts. Margin-top: 0 and margin-bottom: 1rem are also set on all
